# Aurora 极光背景特效

## 已集成

Aurora 极光特效已成功集成到项目主背景中。

## 特性

- **视觉效果**: 流动的极光带，类似极地极光效果
- **技术实现**: WebGL (OGL 库)
- **性能**: 使用 GPU 加速，流畅运行
- **颜色主题**: 紫色系渐变 (#667eea → #764ba2)

## 使用位置

- **主页背景**: `Home.vue` - 作为整个主页的背景特效

## 可调参数

```vue
<AuroraBackground 
  :color-stops="['#667eea', '#764ba2', '#667eea']"  // 颜色渐变数组
  :amplitude="1.0"      // 振幅 (0.5 - 2.0)
  :blend="0.5"          // 混合度 (0.1 - 1.0)
  :speed="1.0"          // 速度 (0.5 - 2.0)
  :intensity="0.8"      // 强度 (0.1 - 1.0)
/>
```

### 参数说明

- **colorStops**: 颜色渐变数组，支持 3 个颜色值
  - 默认: `['#667eea', '#764ba2', '#667eea']` (紫色系)
  - 可尝试: `['#7cff67', '#171D22', '#7cff67']` (绿色系)
  
- **amplitude**: 控制极光的波动幅度
  - 值越大，波动越明显
  - 推荐范围: 0.8 - 1.5

- **blend**: 控制极光的边缘混合
  - 值越大，边缘越柔和
  - 推荐范围: 0.3 - 0.7

- **speed**: 控制极光流动速度
  - 值越大，流动越快
  - 推荐范围: 0.5 - 2.0

- **intensity**: 控制极光亮度
  - 值越大，越亮
  - 推荐范围: 0.6 - 1.0 (背景建议 0.7-0.8)

## 性能优化

- 使用 WebGL 硬件加速
- 自动响应窗口大小变化
- 组件卸载时自动清理资源
- 固定定位，不影响页面滚动性能

## 注意事项

1. **依赖**: 需要 `ogl` 库 (已安装)
2. **浏览器支持**: 需要支持 WebGL 的现代浏览器
3. **性能**: 在低端设备上可能需要降低 intensity 或 amplitude
4. **背景色**: body 背景色已设置为深色 (#0a0a0f)，以突出极光效果

## 故障排除

如果极光不显示：
1. 检查浏览器是否支持 WebGL
2. 检查控制台是否有错误
3. 确认 `ogl` 库已正确安装
4. 检查容器元素是否正确挂载
